## What Was Done

### 1. 1h Extended Backtest (`/api/backtest/run-test?interval=1h`)

Yahoo Finance supports up to 720 days of 1h historical bars for NQ=F (futures). The test endpoint now supports `lookback_days` up to 720 and `interval=1h`.

**Three bugs required fixing for 1h mode to detect any trades:**

**Bug 1 — Overnight VWAP distortion**
- 1h bars span midnight→midnight ET (23 bars/day). Computing VWAP from midnight makes 9:30 AM price appear below VWAP because early overnight bars drag the average.
- Fix: Added `session_start_mins: int = 0` param to `_calc_vwap_at`. Pass `570` (9:30 AM in minutes) for 1h mode to exclude overnight bars.

**Bug 2 — iFVG proximity check too tight for 1h**
- At 1h resolution, FVG sizes are 14–95pts but price can be 100–300pts away from them. Old formula `size * 3 + min_st` produced limits of ~92pts → BLOCKED all zones.
- Fix: Added `fvg_prox_mult: float = 3.0` param to `_find_killzone_setup`. Pass `12.0` for 1h mode.

**Bug 3 — ATR minimum stop too large for 1h**
- 1h ATR ≈ 135pts. `atr_dist = 2 × 135 = 270pts` always exceeded the 120pt ATR stop cap, killing all setups even when sweep_dist was only 60–80pts.
- Fix: Added `atr_min_mult: float = 2.0` param. Pass `0.0` for 1h mode (use only `max(sweep_dist, min_stop_pts)` — no ATR floor).

**Additional 1h adjustments:**
- `min_kz_bars=2` (1h bars land at :00, so killzone 9:30–11:30 catches only 10:00 and 11:00 bars)
- `sweep_max_age_mins=180` (1h resolution means sweeps can be 1–2 bars old)

### 2. Korabi Trades Experimental Flags

Four flags added to `run_backtest_test` endpoint, each implementing a concept from Coach Korabi's transcripts:

| Flag | Concept | Implementation |
|------|---------|---------------|
| `exp_tight_stop=1` | Hard cap stop ≤50pts MNQ | Skip setup if `stop_dist > 50` |
| `exp_ext_liq=1` | IRL→ERL: require external sweep | Check `sweep_label` contains EQH/EQL/session/Asia/London/week/swing/BSL/SSL |
| `exp_smt_confirm=1` | SMT divergence NQ+ES | Download ES=F, check if primary makes new 10-bar high while ES fails (or vice versa) |
| `exp_bpr_entry=1` | BPR zone at entry | Detect overlapping bull+bear FVGs; entry must fall inside the overlap zone |

**BPR implementation note:** Had to call `detect_fvg()` directly (not via `get_ict_analysis()`) and use correct type strings `"bullish_fvg"` / `"bearish_fvg"` — the analysis wrapper doesn't expose the raw FVG list.

## Results

### 720-Day Baseline (NQ=F, 1h, no flags)
- **60 trades, 68.3% WR, PF 3.46**
- Net: $6,440 @1ct | $19,320 @3ct over 2 years
- Monthly average: $564 @3ct

### Korabi Flag Results

**`exp_tight_stop=1`** — VALIDATED
- 18 trades over 720 days (0.75/month)
- **83.3% WR, PF 7.25**
- Key insight: When a setup has a natural stop ≤50pts MNQ, it wins 83% of the time.
- Strategy implication: Don't trade ONLY tight-stop setups — take all Arlennys setups (12–15/month at 75% WR) and size up 2× on the ones with natural stops ≤50pts.

**`exp_ext_liq=1`** — Minor improvement
- ~35 trades, 58.8% WR, PF 2.44
- Slightly better quality but not transformative. Correct concept but hard to detect from Yahoo data alone.

**`exp_smt_confirm=1`** — 0 trades (expected)
- NQ and ES are ~0.99 correlated at 1h resolution. Divergence never materializes.
- SMT is a sub-5m concept — requires seeing individual ticks/1m bars where NQ and ES momentarily disagree.

**`exp_bpr_entry=1`** — 0 trades (expected)
- Overlapping unfilled FVGs are extremely rare at 1h (FVGs fill quickly within 1–2 bars).
- BPR is a 5m concept.

---

## Key Findings

1. The 1h Arlennys baseline (720 days) is solid at 68.3% WR — validates the model across 2 full years including normal and volatile markets.
2. **Korabi tight stop (≤50pts)** is the one flag that works at 1h and produces statistically meaningful results: 83.3% WR over 18 trades.
3. SMT and BPR are strictly sub-hourly concepts — the 5m backtest is the right environment to test those.
4. The 3-day volatility regime filter correctly suppressed trading during the April–May 2026 tariff crash period (10+ consecutive blocked days in both 5m and 1h modes).
